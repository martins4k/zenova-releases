# Zenova Optimizer — Releases

Public distribution repo for Zenova Optimizer. **No source code lives here** — the
application source is in a private repository. This repo only hosts:

- **GitHub Pages**: the update-check manifest (`updates/windows-x64.json`), read by
  `UpdateService` inside the app.
- **GitHub Releases**: signed installer downloads (`ZenovaOptimizerSetup-<version>.exe`).

## Update manifest

Served at:

```text
https://martins4k.github.io/zenova-releases/updates/windows-x64.json
```

Fields: `version`, `downloadUrl` (HTTPS, required), `sha256`, `mandatory`,
`minimumSupportedVersion`, `notes`. Generated by
`scripts/release/make-update-manifest.ps1` in the main app repo — see that repo's
`release/README.md` for the full release flow.

## Publishing a new version

1. Build and sign the installer in the app repo.
2. Create a GitHub Release here tagged `vX.Y.Z`, upload the installer as an asset.
3. Run `make-update-manifest.ps1` to regenerate `updates/windows-x64.json` with the new
   version, download URL, and SHA-256.
4. Commit and push — GitHub Pages redeploys automatically.
